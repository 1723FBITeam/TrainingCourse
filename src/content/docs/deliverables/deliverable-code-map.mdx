## 🗺️ Code Map

A **Code Map** is a one-page visual reference that shows every important file in the robot project and what it does. Use this as a quick-reference sheet when navigating the codebase.

### Robot Project Structure

Browse the full project on GitHub: <GitHubLink client:load path={robotConfig.projectRoot} isFolder label="frc/robot/" />


```
frc/robot/
├── Main.java                  — Entry point; starts the robot program
├── Robot.java                 — Handles the robot lifecycle (init, periodic, teleop, auto)
├── RobotContainer.java        — Wires subsystems to controller buttons and selects autos
├── Constants.java             — Stores configuration values (CAN IDs, speeds, limits)
├── LimelightHelpers.java      — Utility methods for Limelight vision camera
├── Telemetry.java             — Sends data to the driver dashboard
├── TrajectoryCalculations.java — Math for shooting trajectory
│
├── subsystems/
│   ├── IntakeSubsystem.java           — Picks up game pieces with rollers and deploy mechanism
│   ├── ShooterSubsystem.java          — Launches notes with dual flywheels, feeder, and hood
│   ├── TurretSubsystem.java           — Rotates the shooter left/right to aim
│   ├── ClimberSubsystem.java          — Handles endgame climbing with elevator and servos
│   └── CommandSwerveDrivetrain.java   — Swerve drive for omnidirectional movement
│
├── commands/
│   └── AutoShootCommand.java  — Coordinates turret aiming, spin-up, and note feeding
│
└── generated/
    └── TunerConstants.java    — Auto-generated swerve config from CTRE Tuner X (read-only!)
```
